My Certifications

SheCodes Basics

SheCodes Basics is an introduction to front-end development and development tools.You'll learn the basics of HTML and how to create an HTML page. You'll learn how to style an HTML page with CSS and how to target certain design elements in your code. You'll learn the workings of JavaScript and how to animate certain elements of your code. You'll become familiar with using popular coding technologies such as Visual Studio Code.

SheCodes Plus

SheCodes Plus covers advanced front-end development and development tools. You'll go through advanced front-end technologies such as HTML, CSS, and Bootstrap. You'll learn about advanced JavaScript techniques (ES6) so you can build your first interactive application. You will learn how to integrate a live API to start building real applications. You will learn how to use Git and GitHub to manage your code the way professionals do with a streamlined development workflow.

Salesforce Marketing Cloud

The Salesforce Marketing Cloud Email Specialist credential is designed for those who can demonstrate knowledge, skills, and experience in email marketing best practices. Includes message design, subscriber and data management, inbox delivery, and external integrations with the Marketing Cloud email application.
